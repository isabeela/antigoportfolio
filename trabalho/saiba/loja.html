Loja Virtual

Loja de Compras Virtual

Loja de compras virtuais, com vários tipos de produtos fictícios.

Assim como os produtos, o dinheiro também é ficticio. Na página inicial, há uma breve explicação de como funciona a Loja virtual. Se for a primeira vez, o usuário se cadastra e recebe 400 coins para comprar os produtos disponíveis. A cada 500 coins, desbloqueia cerca de 5 produtos. Para conseguir coins, o usuário poderá comprar com dinheiro real a quantidade de coins desejada, ou esperar pela recompensa semanal de login que disponibiliza de 50 a 100 coins.

Linguagens utilizadas: Haskell,HTML,CSS,JavaScript e jQuery
